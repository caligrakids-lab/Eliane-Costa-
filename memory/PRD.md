# PRD - Landing Page Dra. Eliane Costa

## Original Problem Statement
Landing page de luxo para Dra. Eliane Costa - Nutricionista Esportiva com paleta de cores marrom/dourado, foto com degradê, efeitos de flutuação e linha brilhante premium.

## User Personas
- **Clientes potenciais**: Pessoas interessadas em nutrição esportiva, emagrecimento
- **Público feminino 40+**: Mulheres que buscam cuidado com saúde e alimentação

## Core Requirements (Static)
- Paleta marrom escuro + marrom claro + dourado/branco (luxo)
- Foto de capa com degradê na parte inferior
- Animação de flutuação na foto
- Linha brilhante dourada movimentando-se abaixo do subtítulo
- Botões com bordas iluminadas e efeito hover
- 4 links: WhatsApp, eBooks, PUNK CROSSFIT, Mounjaro
